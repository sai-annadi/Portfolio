Flask Application - Natural Language Processing

MCQ Bot Using
RAG

The QuickMCQ Bot is a Flask-based application designed to generate multiple-choice questions (MCQs) on various topics such as Java, Python, and Machine Learning. It leverages the Retrieval-Augmented Generation (RAG) approach, using the Mistral 7B VO2 model from Hugging Face, to provide accurate and contextually relevant answers. The bot takes user inputs for the topic and the number of MCQs desired, then uses a FAISS vector store to access and process PDF files containing relevant information. It dynamically constructs MCQs with options and correct answers, presenting them in a user-friendly format. The bot's key components include Flask for the web framework, FAISS for vector storage, and Hugging Face for language model integration. It aims to enhance learning and assessment by providing a quick and easy way to generate MCQs for various topics.
